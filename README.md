# POYOPOYO GAMING - VALORANT公式ホームページ

## 概要

POYOPOYO GAMINGは、VALORANTを愛するプレイヤーたちが集まる最高のゲームコミュニティの公式ホームページです。美しいデザインとインタラクティブな機能を備えたモダンなウェブサイトです。

## 特徴

- 🎮 **VALORANTテーマ**: ゲーミングに特化したデザイン
- 📱 **レスポンシブデザイン**: モバイル、タブレット、デスクトップに対応
- ✨ **モダンなアニメーション**: スムーズなスクロールアニメーションとインタラクション
- 🎯 **インタラクティブ要素**: ホバーエフェクト、リップルエフェクト、カウンターアニメーション
- 📧 **お問い合わせフォーム**: バリデーション機能付き
- 🌈 **イースターエッグ**: コナミコードで秘密の機能を発動

## セクション構成

1. **ヒーローセクション**: インパクトのあるメインビジュアル
2. **私たちについて**: コミュニティの紹介と統計情報
3. **メンバー紹介**: チームメンバーのプロフィール
4. **イベント情報**: 定期的なゲームイベントの案内
5. **お問い合わせ**: 参加希望者向けのフォーム

## 技術仕様

- **HTML5**: セマンティックなマークアップ
- **CSS3**: モダンなスタイリングとアニメーション
- **JavaScript (ES6+)**: インタラクティブな機能
- **Font Awesome**: アイコンライブラリ
- **Google Fonts**: Orbitron（ゲーミングフォント）とRoboto

## ファイル構成


## 使用方法

1. ファイルをダウンロードまたはクローン
2. `index.html`をブラウザで開く
3. ウェブサイトを楽しむ！

## カスタマイズ

### 色の変更
`styles.css`の以下の変数を変更することで、カラーテーマをカスタマイズできます：

```css
/* メインカラー */
--primary-color: #e91e63; /* ピンク */
--secondary-color: #9c27b0; /* 紫 */
--accent-color: #00bcd4; /* 水色 */
```

### コンテンツの編集
`index.html`を編集して、以下の内容をカスタマイズできます：

- チーム名とロゴ
- メンバー情報
- イベントスケジュール
- 連絡先情報
